# Layer Lenses

Clean Python reimplementation and analysis code for COB-ODT / DLGN / ReLU feature-learning experiments.

This repository is being used to migrate legacy notebook-based research code into a reproducible, modular project. The main research setting is data generated by a complete orthogonal balanced oblique decision tree (COB-ODT), with models trained to recover or represent the underlying ODT label structure.

## Repository Layout

- `src/layer_lenses/` — importable Python package with data generation, model definitions, training loops, experiment runners, IO, and analysis helpers.
- `notebooks/` — exploratory notebooks and current analysis work. Notebook code should call into `src/layer_lenses/` when logic becomes reusable.
- `notebooks/old_notebooks/` — legacy reference material from the original projects. Treat as read-only source material.
- `notes/` — research specs, mismatch logs, reproducibility notes, and session logs.
- `tests/` — lightweight regression and validation tests for data generation, training, runners, and summaries.
- `results/` — intended location for generated metrics, plots, tables, checkpoints, and other artifacts.

## Main Code Areas

- `odt.py` generates COB-ODT datasets and exposes utilities for tree traversal.
- `dlgn.py` and `training.py` implement and train DLGN-SF models.
- `relu_mlp.py`, `relu_training.py`, and `relu_analysis.py` implement, train, and analyze standard ReLU MLPs.
- `experiment_config.py`, `experiment_runner.py`, `multiseed.py`, `analysis.py`, and `io.py` support structured multi-seed DLGN experiments and result summaries.

## Quick Start

Install dependencies with the project environment manager:

```bash
uv sync
```

Run tests:

```bash
uv run pytest
```

Use notebooks for exploratory analysis, especially:

- `notebooks/scratch.ipynb` — current ReLU-on-ODT analysis notebook.
- `notebooks/DLGN_multiseed_multipath_runner.ipynb` — DLGN multipath/multiseed exploration.

## Source-of-Truth Notes

- `notes/research_spec.md`
- `notes/research_spec_math.ipynb`

- `notes/paper_mismatches.md`
